CENTRAL PROCESSING UNIT

What is a CPU?

The Central Processing Unit (CPU), often referred to as the "brain" or "heart" of a computer, is the primary hardware component responsible for processing instructions and managing data within a computing system. It plays a central role in executing tasks and ensuring the functionality of the operating system and applications.

Parts of the CPU

1. Arithmetic Logic Unit (ALU)

- Performs mathematical operations (e.g., addition, subtraction) and logical operations (e.g., AND, OR)(ALU)

2. Control Unit (CU) (ALU)

- Coordinates the activities of the CPU by decoding instructions and directing data flow between components

3. Registers

- High-speed storage locations within the CPU that temporarily hold data being processed

4. Cache Memory

- A small, fast memory unit within the CPU that stores frequently accessed data to speed up operations
